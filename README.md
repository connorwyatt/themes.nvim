# themes.nvim

Neovim colorschemes.

## Installation

```lua
{ "connorwyatt/themes.nvim" }
```

## Usage

```vim
:colorscheme <name>
```

## Theme families

### Sakura

A coordinated family of light and dark themes that share highlight definitions and accent colors, differing mainly in background, surface, and text.

| Colorscheme | Style |
| --- | --- |
| `sakura-day` | light |
| `sakura-dawn` | light |
| `sakura-sky` | light |
| `sakura-dusk` | dark |
| `sakura-night` | dark |
| `sakura-night-muted` | dark |
| `sakura-midnight` | dark |
| `sakura-midnight-muted` | dark |
| `sakura-afterglow` | dark |

```vim
:colorscheme sakura-night
```
